# arvis-bundlephobia

> arvis workflow to find a JavaScript bundle size

<img src="arvis-bundlephobia.gif" width="895">


## Install

```
$ npm install --global arvis-bundlephobia
```


## Usage

In arvis, type `bundlephobia`, <kbd>Enter</kbd>, and your query.
